# Подключение библиотеки

## Gradle/Grails

```groovy
compile 'com.github.gusenov:sqlite-utils:0.0.1'
```

## Apache Maven

```xml
<dependency>
    <groupId>com.github.gusenov</groupId>
    <artifactId>sqlite-utils</artifactId>
    <version>0.0.1</version>
</dependency>
```
